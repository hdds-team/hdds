# hdds-discovery-server

Standalone Discovery Server for HDDS - enables DDS discovery in cloud/NAT environments.

## Usage

```bash
cargo install hdds-discovery-server
hdds-discovery-server --domain-id 39
```

## Features

- Centralized discovery for DDS participants behind NAT/firewalls
- Cloud and Kubernetes compatible
- Configurable via CLI arguments or environment variables
